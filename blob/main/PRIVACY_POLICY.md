# Privacy Policy

**Last updated: March 2026**

## Overview

This application ("fashion-trend-detector") is a personal research project for analyzing and forecasting fashion trends using publicly available data. It is not a commercial product.

## Data Collection

This application does **not** collect, store, or share any personal information from users.

The application accesses the following external APIs for research purposes only:

- **Pinterest API**: Used to retrieve publicly available trend data (keyword trends, category trends). No personal Pinterest account data is accessed.
- **Google Trends (pytrends)**: Used to retrieve publicly available search trend data.

## Data Usage

All data retrieved from external APIs is used solely for:

- Academic and personal research on fashion trend analysis
- Building and evaluating forecasting models

No data is sold, shared, or used for advertising purposes.

## Data Storage

No user data or personal information is stored by this application.
Trend data retrieved from APIs is used transiently for analysis and is not persisted beyond a single session.

This application uses the following third-party APIs, each governed by their own privacy policies:

- [Google Privacy Policy](https://policies.google.com/privacy)
